# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/RafaelGoulartB/rest-api-mongo.git
```
### üì¶ Run API

```bash
# Go to server folder
$ cd rest-api-mongo

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn run dev
```
Access API at http://localhost:3333/
